# STM32G431 SPWM 逆变器 — 量产调试步骤

---

## 调试前准备

| 设备 | 用途 |
|---|---|
| 可调直流电源 (0~450V) | 模拟母线电压 |
| 万用表 | 测电压/通断 |
| 示波器 (≥100MHz) | 测 PWM 波形和死区 |
| USB-TTL 串口模块 (3.3V) | 接 PB6/PB7 看调试输出 |
| 阻性假负载 (100W~500W) | 负载测试 |
| ST-Link V2 | 烧录/调试 |

---

## 第一步：裸板检查（不接高压）

### 1.1 短路检查
- 万用表蜂鸣档，测所有电源对地是否短路
- 3.3V / 5V / 母线 DC+ / DC- 各自对地

### 1.2 烧录固件
- ST-Link 接 PA13(SWDIO) + PA14(SWCLK) + GND + 3.3V
- 烧录 `build/stm32g431_spwm_inverter.hex`
- 串口接 PB6(TX) + GND，115200-8-N-1，确认收到 `PARAM: defaults loaded`

### 1.3 OLED 检查
- OLED 四针接好（VCC/GND/PB10/PB11）
- 上电后显示 `INVERTER V2.0` 主屏
- 短按 PA12(OK) 进入菜单，UP/DOWN 翻页确认按键正常

---

## 第二步：低压母线调试（DC 30V）

> 用可调电源限流 0.5A，模拟低压母线。**不要直接上 400V！**

### 2.1 确认 PWM 不输出
- 母线给 30V DC
- 串口应显示 `SYSTEM READY, wait relay...`
- 但不显示 `RELAY ON, SPWM SOFTSTART`
- 示波器测 PA8/PA7 应无 PWM 波形 ← **正确！母线 30V 远低于 310V 窗口**

### 2.2 模拟母线电压达标（临时修改）
- 将 `VBUS_RELAY_ON_V` 临时改为 `25`（低于当前 30V）
- 将 `VBUS_RUN_MIN_V` 临时改为 `25`
- 将 `VBUS_RELAY_STARTUP_DELAY_MS` 临时改为 `50`
- 重新编译烧录

### 2.3 检查 PWM 波形
- 上电后 OLED 应显示 `ON` 状态
- 示波器测 PA8(TIM1_CH1) 和 PA7(TIM1_CH1N)：
  - 应为 22kHz 互补方波
  - 死区约 1.0us（两个波形边沿之间的间隙）
  - 占空比随软启动缓慢增大
- 同样检查 PA9(CH2) 和 PB0(CH2N)

### 2.4 检查死区
- 示波器双通道，CH1 接 PA8，CH2 接 PA7
- 光标测 CH1 下降沿到 CH2 上升沿的时间差 ≈ 1.0us
- **如果死区太小甚至重叠 → MOSFET 会直通炸管！**

### 2.5 恢复正常阈值
- 把上一步改的三个值恢复原值，重新编译烧录

---

## 第三步：分压比校准（DC 30V → 400V）

### 3.1 确认分压比
- 万用表实测母线分压电阻：R_upper = ?Ω, R_lower = ?Ω
- 计算分压比 = (R_upper + R_lower) / R_lower
- 修改 `BUS_VOLT_DIV_RATIO` 为实测值

### 3.2 电压读数校准
- 可调电源输出几个电压点（50V / 100V / 150V / 200V），每个点：
  - 读串口或 OLED 上显示的 `Vbus` 值
  - 与万用表实测值对比
- 如果偏差大（>5%），微调 `BUS_VOLT_DIV_RATIO`

---

## 第四步：额定母线调试（400V DC）

> 可调电源设置限流 1A，缓慢升压。

### 4.1 上电时序确认
- 串口输出：
  ```
  PARAM: defaults loaded
  SYSTEM READY, wait relay...
  RELAY ON, SPWM SOFTSTART
  ```
- OLED 显示 Vbus 和 Vout 缓慢上升

### 4.2 输出电压校准（空载）
- 万用表 AC 档测输出端，确认 220V ± 5V
- 若偏差 >5V：进 OLED 菜单调整 `VoutADC`
  - 电压偏高 → 减小 VoutADC
  - 电压偏低 → 增大 VoutADC
  - 保存到 Flash

### 4.3 PID 参数调试（核心）

> PID 参数直接决定稳压精度和动态响应，每块板子首次量产必须调参。

#### 4.3.1 确认 PID 在工作
- 空载时 OLED 显示 `Amp: xxx/1000` 稳定在某个值（非 0，非满幅）
- 加负载 → Amp 值应自动增大 → 电压保持 220V
- 串口每秒输出 `RUN vout=... amp=...`，观察 vout 是否稳定在目标值

#### 4.3.2 调参步骤（阻性负载）

| 步骤 | 操作 | 目标 |
|---|---|---|
| **1. 设 Ki=0** | 菜单将 `PID Ki` 设为 0 | 先调纯比例控制 |
| **2. 调 Kp** | 从默认值 (0.254) 逐渐增大，每次加 0.025 | 电压快速逼近 220V |
| **3. 找临界点** | 继续增大 Kp 直到输出电压出现轻微振荡（±3V 来回摆动） | 找到 Kp 临界值 |
| **4. Kp 减半** | 将 Kp 设为临界值的一半 | 留稳定裕度 |
| **5. 调 Ki** | 从 0 逐渐增大 Ki，每次加 0.005 | 消除稳态误差（220V 无偏差） |
| **6. 加负载验证** | 空载↔满载切换 | 电压波动 < ±5V，恢复时间 < 1s |

#### 4.3.3 验收标准

| 测试 | 标准 |
|---|---|
| 空载电压 | 220V ± 3V |
| 满载电压 | 220V ± 5V |
| 空载→满载瞬态 | 电压跌落 < 15V，恢复 < 500ms |
| 满载→空载瞬态 | 电压过冲 < 20V，恢复 < 500ms |
| 稳态振荡 | 无（Amp 值稳定不摆动） |

#### 4.3.4 快速调参技巧
- **响应太慢**（加载后电压恢复慢）→ 增大 Kp
- **稳态有偏差**（空载 218V 达不到 220V）→ 增大 Ki
- **电压振荡**（±5V 以上来回摆动）→ 减小 Kp 或 Ki
- **启动过冲**（软启动结束时电压冲过 220V）→ 减小 Kp，增大 `St.Limit`
- 每调一个参数后保存到 Flash，断电再上电确认

### 4.5 50Hz/60Hz 切换
- PB3 接 GND → OLED 显示 50Hz
- PB3 接 3.3V → OLED 显示 60Hz
- 万用表频率档确认

## 第五步：保护功能验证

### 5.1 母线欠压保护
- 缓慢降低母线电压
- 当 Vbus < 310V → OLED 显示 `FAULT: 4` (UNDER_BUS)
- PWM 停止输出

### 5.2 母线过压保护
- 缓慢升高母线电压
- 当 Vbus > 450V → OLED 显示 `FAULT: 6` (OVER_BUS)
- PWM 停止输出

### 5.3 过温保护
- 用热风枪吹 NTC 或短接 NTC 到高温模拟
- 当温度 > 60.0℃ → OLED 显示 `FAULT: 3` (OVER_TEMP)
- 风扇应在 40℃ 自动开启

### 5.4 短路保护（硬件 BKIN）
- **仅在生产治具上进行！**
- 用低阻值功率电阻瞬间短路输出
- BKIN 应纳秒级关断 PWM
- 串口打印 `FAULT 1 SHORT`
- 2 秒后自动重试 → 3 次后永久锁定

### 5.5 过载保护
- 接假负载，逐步加大
- OLED 上 `P:` 显示的功率持续超过设定值时
- 延时后触发 `FAULT: 2` (OVERLOAD)

---

## 第六步：带载测试

### 6.1 阻性负载
- 100W 灯泡 → 220V 输出正常，功率读数合理
- 500W 加热器 → 输出电压稳定

### 6.2 感性负载（如有）
- 小功率电机/风扇 → 启动不误报短路
- 观察启动阶段电流不触发过载

---

## 第七步：出厂设置

### 7.1 恢复默认参数
- 上电时按住 OK 键 → 加载编译期默认值
- 或进菜单逐项确认参数与生产规格一致

### 7.2 保存参数
- 菜单 → SaveFlash → YES → OK
- 断电 → 重新上电 → 确认参数保持

### 7.3 最终检查清单

| 检查项 | 标准 |
|---|---|
| 空载输出电压 | 220V ± 5V AC |
| 输出频率 | 50.0 / 60.0 Hz |
| 死区 | ≥ 1.0 us |
| 母线欠压保护 | < 310V 关断 |
| 母线过压保护 | > 450V 关断 |
| 短路保护 (BKIN) | 纳秒关断 |
| OLED 显示 | 主屏信息完整 |
| 按键菜单 | 进入/编辑/保存正常 |
| 串口输出 | 115200 正常打印 |
| 故障 LED | 故障时闪烁对应次数 |

## 快速诊断

| 现象 | 可能原因 |
|---|---|
| 上电无显示 | 3.3V 供电 / OLED I2C 接线 / 固件未烧录 |
| 一直显示 STOP | 母线电压未达 310V / 分压比不对 |
| PWM 无输出 | TIM1 引脚复用冲突 / MOE 未使能 |
| 输出波形失真 | 死区设置不对 / LC 滤波器参数不匹配 |
| 频繁报 OVERLOAD | 电流标定系数偏大 / 过载阈值偏低 |
| 一上电就短路保护 | BKIN 引脚悬空（需下拉）/ 比较器阈值太低 |
| OLED 花屏 | I2C 速率过高 / 线太长 / 上拉电阻太大 |

---

> 改 `BUS_VOLT_DIV_RATIO` 适配不同分压比，改 `VOLT_RMS_FACTOR` / `CURR_RMS_FACTOR` 适配不同传感器。其余参数通过 OLED 菜单调节即可。
