# STM32G431 纯正弦波后级 SPWM 驱动板程序

本目录是在上级 STM8S 工程基础上新增的 STM32 HAL 版参考工程。建议 MCU 选 `STM32G431KBT6/KBU6`，32 脚封装，有 TIM1 互补 PWM、BKIN 硬件刹车、快速 ADC/DMA，适合 22 kHz 纯正弦逆变后级。

## 已实现功能

- TIM1 22 kHz 互补 SPWM，支持单极性和双极性模式。
- 50 Hz / 60 Hz 输入脚选择。
- 死区时间集中配置，默认 1 us。
- 软启动时间集中配置，默认 3 s。
- 输出电压、输出电流、温度、母线电压 4 路 ADC DMA 采样和滤波。
- 工业常用固定点 PID，带积分限幅、测量微分接口和输出斜率限制。
- 短路保护：外部比较器接 TIM1 BKIN，硬件立即关断，同时短路信号输入 MCU。
- 短路后 2 s 自动重试，循环 3 次后锁定关闭 SPWM。
- 过载、过温、母线欠压/过压、输出低压保护。
- LED 按闪烁次数区分故障，串口 115200 打印故障码和运行状态。

## 关键引脚

- `PA8` TIM1_CH1：A 桥臂高侧。
- `PA7` TIM1_CH1N：A 桥臂低侧。
- `PA9` TIM1_CH2：B 桥臂高侧。
- `PB0` TIM1_CH2N：B 桥臂低侧。
- `PA6` TIM1_BKIN：硬件短路比较器输出，高电平触发。
- `PB4`：短路信号送 MCU，用于软件状态机重试计数。
- `PA0..PA3`：输出电压、电流互感器、温度、母线电压 ADC。
- `PB1`：50/60 Hz 选择，高电平 60 Hz。
- `PB2`：SPWM 模式选择，高电平双极性，低电平单极性。
- `PB6/PB7`：USART1 TX/RX，115200。
- `PB8/PB9/PB5`：故障 LED、风扇、继电器。

实际 PCB 必须按封装复核复用功能，必要时用 CubeMX 重新分配引脚。

## 编译

这个目录只放应用代码和构建脚本，没有内置 ST HAL/CMSIS 库。生成 HEX 需要：

1. 安装 GNU Arm Embedded Toolchain，使 `arm-none-eabi-gcc` 在 PATH 中。
2. 从 STM32CubeG4 或 CubeMX 工程复制：
   - `Drivers/CMSIS`
   - `Drivers/STM32G4xx_HAL_Driver`
3. 在本目录运行：

```bat
build.bat
```

成功后输出：

```text
build\stm32g431_spwm_inverter.hex
```

也可以把 `Core`、启动文件和链接脚本导入 STM32CubeIDE 后生成 HEX。

## 专用 SPWM 芯片参考要点

EG8010/EGS002 一类专用正弦逆变方案通常提供 50/60 Hz、死区选择、软启动、过流/过温/欠压保护和故障输出。本工程按这些成熟方案的思路实现为可调软件参数，但短路仍必须交给硬件比较器和 TIM1 BKIN 先关断，MCU 只负责记录、延时重试和锁定。

## 感性负载建议

- 短路阈值用硬件比较器设为真正危险峰值，过载阈值用电流互感器 RMS/整流采样并加入延时，不要把电机启动浪涌当短路。
- PID 输出有斜率限制，启动阶段过流时主动压低调制度，减少变压器/电机励磁冲击。
- 输出采样建议使用隔离或高阻分压后整流滤波，PID目标值必须用真有效值表或实测标定修正。
- LC 滤波器、电流互感器相位延迟和采样滤波会影响负载调整率，带电机时优先调小 Ki、限制最大调制度，再逐步提高响应。
